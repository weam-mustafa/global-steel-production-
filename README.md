Global Steel Production Dashboard (2024–2025)
 Project Overview
This Power BI dashboard provides a global analysis of steel production trends for 2024–2025, highlighting production volumes, regional performance, and year-over-year growth patterns across major steel-producing countries.
The dashboard is designed to support data storytelling, enabling users to quickly identify leading producers, regional trends, and countries experiencing growth or decline.
Objectives
Compare steel production levels across major countries
Analyze regional performance trends (Asia, Europe, Americas, Africa)
Identify countries with positive and negative year-over-year growth
Present insights through an interactive and visually engaging dashboard

 Key Features

Bar Chart: Top steel-producing countries by production volume

Growth Analysis Chart: Year-over-year production change by country

Interactive World Map: Geographic distribution of steel production

Insight Summary: Key observations highlighting regional performance
Key Insights

Asia leads global steel production with consistent year-over-year growth

Europe shows mixed performance, with variations across countries
Africa and South America remain relatively stable with minor fluctuations
China remains the dominant producer, followed by India and Japan
